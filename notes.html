The Web

The Web (or World Wide Web) is a collection of documents and other files.Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.

Components of HTML:

  • Text that the user will actually read in their browser
  • Markup which tells the browser what that that text should look like as well as its arrangements
  • References to include other documents and files such as images and videos
